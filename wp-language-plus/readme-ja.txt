管理画面からWordPressの言語パッケージが追加できるようになります。インストールされている言語は、ユーザー毎に管理画面の表示言語を選択できるようになります。

== 説明 ==

管理画面からWordPressの言語パッケージが追加できるようになります。インストールされている言語は、ユーザー毎に管理画面の表示言語を選択できるようになります。

== インストール方法 ==

1. プラグイン一覧ページで、**WP Language Plus** を有効化します。

== 使い方 ==
= 言語の追加 =
1. [ツール]メニューに追加されている[言語]メニューをクリックすると言語追加ページに遷移します。
2. 追加したい言語にチェックを入れて「インストール」ボタンをクリックしてください。インストールが完了すると、インストールされた言語が表示されます。

= ユーザー別表示言語の変更 =
1. ユーザーの編集ページ、もしくはプロフィールページに管理画面の言語選択が追加されていますので、希望する言語を選択の上、更新を行って下さい。

※ plugins_loaded フックよりも前のタイミングで翻訳ファイルの読み込みを行っているプラグインの表示言語は変更されません。
