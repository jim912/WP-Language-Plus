== Description ==

Add languages of WordPress from Dashboard. Every user can choose dashboard language on Profile page.

== Installation ==

1. Activate **WP Language Plus** through the 'Plugins' menu in WordPress.

== Usage ==
= Add Languages =
1. Click 'Languages' menu. It's child of 'Tools'.
2. Check languages you want add. Click 'Install' button. When complete install, displayed update message on panel.

= user's language =
1. Added language dropdown list on Profile and User Edit page. Choose language and update user.

note : Disable user's language for plugins that loading localize file before 'plugins_loaded'
